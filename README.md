# Lift Traffic Analysis Tool

A Python-based automation tool developed during my internship at Larsen & Toubro (L&T) for lift traffic analysis calculations.

The original workflow was based on an Excel sheet containing multiple engineering formulas and input parameters. The formulas and calculation logic were studied, decoded, and converted into an automated Python application.

---

## Project Overview

This project automates lift traffic analysis calculations by:

- Taking user input parameters
- Processing engineering calculation formulas
- Generating calculated traffic analysis outputs instantly

The application was initially developed as a desktop application using Tkinter and later migrated to a web-based interface using Streamlit for easier deployment and accessibility.
